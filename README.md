# Next.js Tetris ゲーム

## 🎮 プロジェクト概要
このプロジェクトは、[ヘロンの数学ちゃんねる様](https://www.youtube.com/@heron-no-suugaku)の[元ゲーム会社のプログラマー、テトリス何分で作れる？【JavaScriptプログラミング】](https://youtu.be/34uAtYSirWk?si=SnMOEXyCq2gKJb4P) を参考に作成されたテトリスゲームです。

## 🚀 技術スタック
- **フレームワーク**: Next.js
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
